About

This project analyzes NCAA basketball data with the Python 3.x sportsreference API. We have built several Machine Learning Models to predict the outcomes of NCAA regular season and March Madness basketball tournament games. In order to make matchup predictions, we pre-processed boxscore data to calculate the moving averages of the statistics for each team prior to playing in a matchup. All models were trained using these moving averages.

Testing Results

Testing results were trained and measured on a partitioned dataset of the 2020 NCAA Men's Basketball season. Future implementations will analyze larger dataset with data from the 2010 - 2021 seasons.
